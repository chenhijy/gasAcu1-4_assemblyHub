Description

LiftOver (Hinrichs et al. 2006) chains (Kent et al. 2003) between gasAcu1 and gasAcu1-4 were generated from BLAT (Kent 2002) alignments using the script doSameSpeciesLiftOver.pl as well as Kent Utilities following the procedure described here: http://genomewiki.ucsc.edu/index.php/DoSameSpeciesLiftOver.pl.
